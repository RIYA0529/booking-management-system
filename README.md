# 🏨 Booking Management System (Laravel)

A simple web-based Booking Management System built using Laravel.  
This project helps users manage bookings easily with CRUD operations.

---

## 🚀 Features

- Add new bookings
- View all bookings
- Update booking details
- Delete bookings
- Simple and clean UI

---

## 🛠️ Tech Stack

- Laravel (PHP Framework)
- PHP
- MySQL
- HTML
- CSS
- Bootstrap
